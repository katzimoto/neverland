### ⚠️ Warnings (should fix)

1. **Feature flag not enforced in endpoints** (`src/services/api/main.py`)
   - `feature.annotations` exists in `system_config` but endpoints do not check it.
   - Consistent with existing features; fix globally in follow-up.

2. **SQLite JSON stored as string** — `_parse_json` helper added in API layer to handle SQLite's string JSON storage. PostgreSQL native JSON works without this. The helper is harmless for Postgres.

3. **Hard delete vs soft delete** — Annotations use hard delete (per spec), unlike comments which use soft delete. This is intentional but worth noting for audit requirements.
