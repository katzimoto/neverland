# Review Reports

Per-PR review reports written by the Reviewer agent.

## Format

Each file follows the template in `AGENTS.md` §4.

## Index

| PR | File | Phase | Verdict |
|----|------|-------|---------|
| #11 | `review/11.md` | 05a Preview service | Approved |
| #12 | `review/12.md` | 05b Translation enrichment | Approved |

## Naming convention

```
review/<pr-number>.md
```

For PRs with multiple review rounds, append `-r<N>`:
```
review/13-r1.md   # first review
review/13-r2.md   # second review (after fixes)
```
