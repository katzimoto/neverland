### What changed

1. **RAG service:** `src/services/rag/service.py`
   - Encodes questions, retrieves Qdrant chunks, builds context, calls Ollama, and returns citations.
   - Falls back to retrieved passages when Ollama generation fails.

2. **API endpoint:** `src/services/api/main.py`
   - Adds `POST /qa` with `question` and `top_k`.
   - Reads `llm.qa_system_prompt` from `system_config`.
   - Enforces `feature.rag_qa` from both settings and runtime system config.

3. **Models and tests**
   - Adds `QuestionRequest`, `AnswerResponse`, and `Citation`.
   - Adds eight integration tests for the new endpoint.

4. **Docs**
   - Updates `CHANGELOG.md` and `agents.md`.

5. **Permission indexing**
   - Indexes ES/Qdrant permission payloads from source grants instead of a placeholder group.
